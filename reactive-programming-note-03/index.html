有状态的对象

一般会把世界描绘成一系列的对象，它们中的一些的状态会随着时间而改变。称一个对象有状态是指它的行为受到其历史的影响。

比如一个银行账号是有状态的，因为它对“能取100元吗”这个问题的回答在此账号的生命周期中是有可能变化的。
